# Атрибуция и государственная символика

Этот документ описывает статус государственной символики Кыргызской
Республики в проекте «Теңдик».

## Текущий статус: официальные символы НЕ включены

В репозитории **нет** официального флага и **нет** официального герба
Кыргызской Республики. В интерфейсе используются только **стилизованные
декоративные мотивы** собственной отрисовки, которые не изображают
официальные государственные символы и нигде не подписываются как флаг
или герб:

- `src/components/decor/tunduk-mark.tsx` — стилизованный түндүк;
- `NationalMotifChip`, `NationalBadge` (`src/components/decor/kyrgyz-flag.tsx`)
  — красно-золотой декоративный мотив түндүк с подписью «Кыргызстан»;
- `src/components/decor/kyrgyz-ornament.tsx`, `ethnic-divider.tsx` —
  орнаментальные мотивы (шырдак/курак, кочкор мүйүз);
- `src/components/decor/ala-too-silhouette.tsx` — силуэт гор Ала-Тоо.

Ранее использовавшаяся самодельная SVG-реконструкция флага удалена:
приблизительную геометрию нельзя выдавать за официальный государственный
символ.

## Почему официальный asset ещё не добавлен

Сборочное окружение, в котором готовился редизайн, имеет ограниченную
сетевую политику: обращения к допустимым источникам официального флага
(официальные государственные ресурсы КР, Wikimedia Commons) блокируются
сетевым прокси (HTTP 403). Загрузить и проверить официальный файл из этого
окружения невозможно, а использование непроверенных зеркал (случайные
GitHub-репозитории, npm-пакеты, CDN) запрещено правилами проекта.

## Как добавить официальный флаг

1. Скачайте официальный векторный файл флага из проверенного источника,
   например:
   - страница файла на Wikimedia Commons:
     `https://commons.wikimedia.org/wiki/File:Flag_of_Kyrgyzstan.svg`
     (убедитесь, что файл соответствует действующей редакции флага —
     с учётом изменений, принятых в декабре 2023 года, — и что на
     странице файла указано происхождение и статус государственного
     символа / общественного достояния);
   - либо официальные государственные ресурсы Кыргызской Республики.
2. Сохраните файл БЕЗ изменений геометрии, пропорций, viewBox и цветов в:
   `public/national-symbols/kyrgyzstan-flag.svg`
3. Дополните этот документ: точный URL источника, дата получения,
   автор/происхождение файла (если указаны), лицензия/статус.
4. Подключите флаг в UI через готовый компонент `KyrgyzFlag`
   (`src/components/decor/kyrgyz-flag.tsx`) — он отображает именно этот
   локальный файл с корректным alt для скринридеров.

## Герб

Официальный герб Кыргызской Республики не включён по той же причине.
Самодельная или приблизительная версия герба не создавалась и создаваться
не должна. При появлении проверенного официального SVG действуйте по той
же схеме: локальный файл в `public/national-symbols/`, запись в этом
документе, корректный alt, без искажений и без использования герба как
фона, паттерна или watermark.

## Прочие внешние ресурсы

- Иконки: [lucide-react](https://lucide.dev) (ISC), установлен как
  npm-зависимость проекта.
- Все остальные SVG и паттерны в `src/components/decor/` и
  `src/app/globals.css` — оригинальная отрисовка для этого проекта.
