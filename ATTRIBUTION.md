# Атрибуция и государственная символика

Этот документ описывает статус государственной символики Кыргызской
Республики в проекте «Теңдик».

## Текущий статус: официальные символы отсутствуют

Официальный флаг и официальный герб Кыргызской Республики в UI сейчас
**отсутствуют**: в репозитории нет их файлов, и в production-коде нет
компонентов, которые пытались бы их загрузить. В интерфейсе используются
только **стилизованные декоративные мотивы** собственной отрисовки,
которые официальные государственные символы не изображают и нигде так не
подписываются:

- `src/components/decor/tunduk-mark.tsx` — стилизованный түндүк;
- `src/components/decor/national-motif.tsx` — `NationalMotifChip` и
  `NationalBadge`: красно-золотой декоративный мотив түндүк с подписью
  «Кыргызстан»;
- `src/components/decor/kyrgyz-ornament.tsx`, `ethnic-divider.tsx`,
  `shyrdak-corner.tsx` — орнаментальные мотивы (шырдак/курак, кочкор мүйүз);
- `src/components/decor/mountain-range.tsx` — силуэт гор Ала-Тоо;
- `src/components/decor/boz-uy.tsx` — иллюстрация боз үй (юрты);
- `src/components/decor/ethnic-texture.tsx` — переиспользуемые слои
  текстур (войлок/орнамент/кереге).

Ранее существовавшая самодельная SVG-реконструкция флага удалена:
приблизительную геометрию нельзя выдавать за официальный государственный
символ. Также удалена неиспользуемая заготовка компонента, ссылавшаяся
на ещё не добавленный файл.

## Внешние изображения (фото/растр): не добавлялись

Второй этап редизайна предполагал возможность добавить 1–3 фотографии
из открытых источников (горы Кыргызстана, юрта, войлок, шырдак) под
свободными лицензиями (CC0/CC BY/CC BY-SA, Wikimedia Commons и т.п.).

В сборочном окружении сетевой доступ к допустимым источникам изображений
заблокирован политикой прокси (HTTP 403 для Wikimedia/officialных
ресурсов), а непроверенные зеркала (случайные сайты, Pinterest, стоки
без лицензии, GitHub-assets) использовать нельзя. Поэтому внешние
растровые изображения **не добавлялись**.

Вместо фотографий использованы **оригинальные локальные SVG-композиции**,
нарисованные для этого проекта (горы Ала-Тоо со снежными вершинами, боз
үй, орнаменты, текстуры войлока/кереге). Это лёгкие, легально чистые
ассеты без сторонних правообладателей и без layout shift. Когда появится
проверенное открытое фото с ясной лицензией, его можно добавить локально
в `public/` отдельным PR с записью в этом документе (URL источника, автор,
лицензия, дата, факт изменения файла).

## Как будут добавляться официальные символы

Официальные флаг и/или герб добавляются **отдельным PR** после ручной
проверки источника:

1. Скачать официальный векторный файл из проверенного источника,
   например:
   - страница файла на Wikimedia Commons:
     `https://commons.wikimedia.org/wiki/File:Flag_of_Kyrgyzstan.svg`
     (убедиться, что файл соответствует действующей редакции флага —
     с учётом изменений, принятых в декабре 2023 года, — и что на
     странице файла указано происхождение и статус государственного
     символа / общественного достояния);
   - либо официальные государственные ресурсы Кыргызской Республики.
2. Сохранить файл БЕЗ изменений геометрии, пропорций, viewBox и цветов
   в `public/national-symbols/` (например,
   `public/national-symbols/kyrgyzstan-flag.svg`).
3. Дополнить этот документ: точный URL источника, дата получения,
   автор/происхождение файла (если указаны), лицензия/статус.
4. Создать в том же PR отдельный компонент, отображающий этот локальный
   файл с корректным alt (для флага:
   «Кыргыз Республикасынын мамлекеттик туусу»), явными width/height
   (без layout shift) и режимом decorative (`alt=""` + `aria-hidden`)
   для чисто декоративных повторов.

До появления проверенных файлов добавлять самодельные, приблизительные
или взятые из непроверенных источников версии государственных символов
нельзя. Внешние hotlink в production не используются. Герб нельзя
использовать как фон, паттерн, watermark или кнопку.

## Прочие внешние ресурсы

- Иконки: [lucide-react](https://lucide.dev) (ISC), установлен как
  npm-зависимость проекта.
- Все SVG и паттерны в `src/components/decor/` и `src/app/globals.css`
  — оригинальная отрисовка для этого проекта.
