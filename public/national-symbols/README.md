# national-symbols

Каталог для проверенных официальных государственных символов Кыргызской
Республики (флаг, герб).

Сейчас каталог пуст: официальные файлы ещё не добавлены. Порядок
добавления, требования к источникам и атрибуция — в `ATTRIBUTION.md`
в корне репозитория.

Ожидаемые файлы:

- `kyrgyzstan-flag.svg` — официальный флаг КР (без изменений геометрии,
  пропорций и цветов); отображается компонентом
  `src/components/decor/kyrgyz-flag.tsx` (`KyrgyzFlag`).
- `kyrgyzstan-emblem.svg` — официальный герб КР (при наличии
  проверенного источника).

Самодельные, приблизительные или непроверенные версии государственных
символов размещать здесь нельзя.
