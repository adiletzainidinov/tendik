# national-symbols

Каталог для проверенных официальных государственных символов Кыргызской
Республики (флаг, герб).

Сейчас каталог пуст: официальные файлы отсутствуют, и в production-коде
нет компонентов, которые пытались бы их загрузить. Официальные assets
будут добавлены **отдельным PR** после ручной проверки источника —
порядок и требования описаны в `ATTRIBUTION.md` в корне репозитория.

При добавлении в том же PR создаются отдельные компоненты с корректными
alt, width и height. Самодельные, приблизительные или непроверенные
версии государственных символов размещать здесь нельзя.
